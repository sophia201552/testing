1、模块头部定义引用的时候，顺序为
三方的公共库 -> 自己的公共库 -> 自己的非公共库
每一块引用之间要换一行

2、在模块自己的 css 中如果想引用全局的 css 样式，做法如下：
假设模块内的 css 的 class 为 .aaa，全局的为 .bbb
:global(.bbb) .aaa{
  color: blue;
}
:global(.bbb):hover .aaa {
  color: red;
}
说明：
.aaa 会被解析器转变成随机字符串（假设被解析成了 .xxx），但是 .bbb 不会，整个文件解析后的结果可能是：
.bbb .xxx {
  color: blue;
}
.bbb:hover .xxx {
  color: red;
}
